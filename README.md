# My Dotfiles

Cloning the files:

**The files will be overwritten, watch out!**

## `.vimrc`

```bash
curl https://raw.githubusercontent.com/math2001/dotfiles/master/.vimrc > ~/.vimrc
```

## `.bashrc`

```bash
curl https://raw.githubusercontent.com/math2001/dotfiles/master/.bashrc > ~/.bashrc
```
